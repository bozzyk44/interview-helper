# Конвенции промптов (answer.py)

Латентность до первого токена — критична. Все правила следуют из этого.

- **Модель:** `haiku` (`claude -p --model haiku`). Не поднимать тир модели ради качества без замера латентности через /latency-check.
- **Системный промпт:** короткий и статичный (кэшируется). Роль: «суфлёр на собеседовании», отвечать от первого лица кандидата, сжато — 2–5 предложений или маркированный список, без преамбул.
- **Контекст разговора:** скользящее окно последних ~2000 токенов транскрипта, помеченное источниками `[interviewer]` / `[me]`. Не отправлять весь транскрипт.
- **Стриминг:** всегда `--output-format stream-json --verbose`; парсить события и выводить текст по мере прихода.
- **Один вопрос — один вызов:** не батчить вопросы; новый вопрос от интервьюера отменяет незавершённый ответ на предыдущий.
- **Язык ответа** = язык вопроса.
