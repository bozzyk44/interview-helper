# Session report

1. Вход: путь к транскрипту из аргумента; иначе возьми самый свежий файл в `sessions/`. Если его нет — сообщи и остановись.
2. Прочитай транскрипт. Если в `context/vacancy.md` есть описание вакансии — учти его при оценке.
3. Сформируй отчёт на русском:
   - **Саммари встречи** — 5–10 предложений: о чём говорили, ключевые темы.
   - **Вопросы и ответы** — таблица: вопрос интервьюера → как ответил кандидат → как стоило ответить лучше (если стоило).
   - **Оценка** — сильные и слабые стороны выступления, вердикт по шкале 1–10 с обоснованием, 3 конкретных рекомендации к следующей сессии.
4. Сохрани отчёт рядом с транскриптом как `<имя-транскрипта>.report.md` и покажи его пользователю.
