# Mock interview

Прогоняет полный пайплайн (транскрипция → детекция вопросов → ответы) на аудиофайле, имитируя живую сессию.

## Шаги

1. Аудиофайл — из аргумента скилла; иначе спроси путь. Поддерживаются WAV/MP3 (faster-whisper декодирует через ffmpeg/PyAV).
2. Запусти `uv run python -m interview_helper.main --input-file <path>` (флаг `--input-file` подменяет живой захват чтением файла; если флага ещё нет — реализуй его в `main.py`: файл читается чанками с той же скоростью подачи, что и реальный звук, источник помечается как loopback).
3. Наблюдай вывод: транскрипт с таймстампами и ответы. По завершении сообщи пользователю, какие вопросы были распознаны и на какие пайплайн ответил.
